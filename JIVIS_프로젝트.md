# JIVIS 개인 프로젝트

> **JIVIS** — 나만의 AI 친구 + 비서 (Just Intelligent, Very Intimate System)

---

## 📌 프로젝트 개요

Python + Claude/Gemini API를 활용해 **나를 기억하고, 공감하고, 도와주는 AI 친구 겸 비서**를 직접 만드는 4주 프로젝트.

단순 챗봇에서 시작해서 → 나를 기억하는 친구 → 도움이 되는 비서 → 음성으로 대화하는 자비스까지 단계별로 완성한다.

---

| 주차 | 목표 | 주요 기능 | 학습 내용 |
|------|------|-----------|-----------|
| **1주차** | 🧠 JIVIS Core 구축 | 채팅 UI, AI 연결, 기본 대화, 이름 기억 | 함수, JSON, 파일 입출력, API 호출 |
| **2주차** | 🤝 친구 AI 만들기 | 장기 기억, 감정 공감, 대화 기록 기반 응답, 페르소나 | Prompt Engineering, Memory 구조, SQLite |

- **AI**: Claude API / Gemini API
- **DB**: SQLite
